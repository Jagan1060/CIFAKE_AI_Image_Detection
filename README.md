# CIFAKE AI Image Detection Using Deep Learning

## Project Overview

This project focuses on detecting AI-generated and real images using Deep Learning techniques. A Convolutional Neural Network (CNN) model was trained on the CIFAKE dataset to classify images as either AI-generated or authentic.

## Problem Statement

With the rapid growth of Artificial Intelligence, highly realistic synthetic images are being generated. These images can be misused for misinformation, fake content creation, and digital manipulation. Therefore, an effective system is required to identify AI-generated images.

## Objectives

* Detect AI-generated and real images.
* Train a CNN model using the CIFAKE dataset.
* Evaluate model performance using standard metrics.
* Generate visual performance graphs and confusion matrix.

## Dataset

* Dataset: CIFAKE Dataset
* Classes:

  * AI Generated
  * Real Images
